# Determine installation media

## What is it?

The SQL Agent installer is delivered through several media types. Throughout the install and upgrade procedures, the documentation uses `<media>` as a placeholder for the delivery method you choose. Pick the option that matches what you have, then substitute that name wherever you see `<media>`.

## Choose your delivery method

Replace `<media>` with the desired method of delivery:

- **USB** — Plug the USB drive into the machine and copy the installation files to a folder on the local hard drive. Run the installation from the hard drive.
- **FTP** — Download the installation files directly to a folder on the hard drive of the machine where the software will be installed. Running from the local hard drive avoids partial downloads or network drops mid-install.
- **DVD-ROM** — Insert the DVD into the drive and run the installation from the DVD. You may also copy the files to the hard drive first.
- **ISO** — Mount the ISO image as a virtual DVD and run the installation from that location.

:::tip
Whichever method you choose, running the installer from a local hard drive is the most reliable option.
:::
