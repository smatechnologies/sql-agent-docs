# Silent mode

## What is it?

Silent mode lets you install the SQL Agent without interactive prompts, which is useful when you need to:

- Roll the agent out to many machines through a deployment tool.
- Run the install from a scripted or automated workflow.
- Repeat an identical install configuration without manual entry.

## Where to find the procedure

The silent install procedure is shared across OpCon components. To learn how to install the SQL Agent in silent mode, refer to the [Silent Mode](https://help.smatechnologies.com/opcon/core/rolling/Files/Installation/Component%20Installations.htm#Silent) section in the **OpCon Installation** online help.
